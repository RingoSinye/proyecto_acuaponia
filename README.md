Proyecto de Monitoreo, Almacenamiento y Visualización de Datos

Este repositorio contiene los documentos, configuraciones y archivos resultantes del proyecto integral de **Acuaponía, desde la automatización y el control** desarrollado como parte del reto académico planteado por la empresa IDOM.

Estructura del Repositorio

El repositorio está organizado en carpetas, cada una correspondiente a uno de los documentos principales desarrollados:

1. **Documento 1 – Almacenamiento de Datos**
Incluye todo lo relacionado con:
- Configuración del flujo de datos desde Node-RED hacia MySQL.
- Creación de la base de datos y tablas.
- Scripts SQL utilizados.
- Capturas y documentación del proceso.
- Verificación final y consulta de datos en Workbench.

2. **Documento 2 – Visualización y Análisis de Datos en Power BI**
Contiene:
- Archivo .pbix
- Conexión a la base de datos
- Visualizaciones creadas
- Documentación del proceso y metodología

3. **Documento 3 – Sistema SCADA para Monitoreo, Supervisión y Análisis de Datos**
Incluye:
- Configuración de Ignition
- Pantallas SCADA
- Scripts, tags y conexiones
- Documentación final del sistema

Objetivo del Repositorio

Reunir de manera organizada todo el material del proyecto, permitiendo:
- Fácil consulta
- Reutilización de configuraciones
- Referencia para futuros proyectos
- Acceso público y transparente al proceso completo
